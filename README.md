# 🌐 Portfolio Website

This project is a personal portfolio website built with Next.js.

It was created to experiment with modern frontend technologies and to showcase projects in a simple and clean layout.

---

## 🚀 Features

- Responsive layout
- Modern UI with Tailwind CSS
- Project showcase structure

---

## 🛠️ Tech Stack

- Next.js
- TypeScript
- Tailwind CSS

---

## 🧠 What I learned

- Basics of Next.js App Router
- Component-based UI structure
- Styling with Tailwind CSS
